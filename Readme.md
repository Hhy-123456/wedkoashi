# 真实天气预报网页应用（零密钥开源接口版）
## 项目介绍
基于HTML5+CSS3+原生JavaScript ES6开发的课程作业天气预报网站，采用全球免费开源Open-Meteo气象接口，**无需注册、无需API密钥**，实现真实城市搜索、浏览器原生定位、实时气温展示、未来5天含降雨预报、本地数据缓存全套功能，PC/移动端响应式适配。

## 技术栈
1. HTML5：语义化header、main、section标签搭建页面结构
2. CSS3：渐变背景、圆角卡片、Flex/Grid布局、媒体查询响应式开发
3. JavaScript ES6+：箭头函数、async/await异步请求、对象解构赋值、数组高阶遍历
4. 网络请求：原生Fetch API调用Open-Meteo开源气象接口
5. Web拓展API：Geolocation地理位置定位API、localStorage本地持久化存储

## 完整功能清单
### 基础必做功能
1. 城市搜索：输入城市中文名/拼音，支持点击按钮、回车键双触发搜索，编写非空输入合法性校验
2. 实时天气展示：展示城市名、日期、当前气温、天气描述、湿度、风速
3. 未来5天天气预报：展示每日星期、天气图标、高低温，**雨天额外标注（降雨）文字提示**

### 加分拓展功能
4. 地理位置定位：调用浏览器Geolocation API获取用户真实经纬度，自动解析城市加载天气
5. 数据持久化：localStorage保存最近搜索城市，页面刷新自动读取缓存

## 运行方式
1. 将代码保存为index.html文件；
2. 使用Chrome/Edge浏览器双击文件直接打开运行，无需部署服务器、无需申请任何密钥；
3. 开启浏览器网站位置权限可使用定位功能，手动输入城市名可搜索全国任意城市天气。

## 核心代码知识点
1. 输入校验：handleSearch函数trim过滤空格，判断空字符串给出错误提示
2. 异步请求：async/await链式调用地理编码接口+气象接口，分步获取数据
3. ES6语法：定位模块 `const {longitude, latitude} = pos.coords` 解构赋值简化变量
4. 降雨判断：根据官方气象编码区间判定雨天，动态添加样式与文字标注
5. 本地存储：localStorage.setItem/getItem实现城市搜索记录持久化
